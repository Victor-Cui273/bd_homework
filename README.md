# bd_homework
Parallel statistics for binary file (sum/min/max)
# Большие данные: теория и практика – Задание №1

## Описание
Реализация программы `calc_data.py` для параллельного вычисления суммы, минимума и максимума 32-битных беззнаковых целых чисел из бинарного файла с **порядком байт big-endian**.

Программа автоматически определяет количество доступных ядер CPU и разбивает файл на блоки для параллельной обработки (многопроцессорность).

## Запуск

### Локальный запуск (требуется Python 3.10+ и numpy)
```bash
pip install numpy
python calc_data.py <путь_к_файлу>
